# Fireoptimizer

Fireoptimizer is a Windows optimization tool designed to improve system performance for gaming and productivity.

## Features
- CPU optimization
- GPU optimization (AMD / NVIDIA / Intel)
- RAM tuning
- Full system optimization
- Game mode enhancements
- Power plan tuning

## Tech
- C# (.NET)
- Windows Forms
- Visual Studio
